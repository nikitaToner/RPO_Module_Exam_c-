# MediTrack Desktop

Система учета медикаментов для локальных аптек.  

## Требования к окружению
1. ОС: Windows 10/11
2. [.NET 8.0 SDK](https://dotnet.microsoft.com/download)
3. Локально установленный **PostgreSQL 15+** (работает на стандартном порту `5432`).
4. Клиент для управления БД (pgAdmin, DBeaver или psql).

##  Локальное развертывание  

### Шаг 1: Подготовка базы данных
1. Установите PostgreSQL, если он не установлен. Запомните пароль пользователя `postgres`.
2. Создайте новую базу данных с именем `meditrack_db`.
3. Выполните скрипт инициализации `init_db.sql` (лежит в корне репозитория) для создания таблицы и добавления тестовых данных

### Шаг 2: Настройка переменных окружения

В корне проекта создайте файл .env
**НЕ В КОРНЕ ,А ГДЕ ЗАПУСКАЕТСЯ ПРОЕКТ**
Заполните его следующими данными (замените YOUR_PASSWORD на ваш реальный пароль от PostgreSQL):

```env
DB_HOST=localhost
DB_PORT=5432
DB_NAME=meditrack_db
DB_USER=postgres
DB_PASSWORD=1234
```
### Шаг 3: Сборка и запуск
Откройте терминал в папке проекта.

Восстановите зависимости: dotnet restore

Соберите проект: dotnet build

Запустите приложение: dotnet run
